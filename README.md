# AI SQL Agent

AI-powered SQL Agent built with LangChain, Llama 3.3 (Groq), and SQLite that converts natural language queries into SQL and returns database insights.

## Features

- Natural Language → SQL conversion
- Automatic schema inspection
- SQL query generation and execution
- Database reasoning using LLMs
- SQLite database integration

## Tech Stack

- Python
- LangChain
- Groq
- Llama 3.3 70B
- SQLite
- SQLAlchemy

## Workflow

1. User asks a question in natural language.
2. SQL Agent inspects the database schema.
3. LLM generates SQL queries.
4. Query executes on SQLite.
5. Results are returned in plain English.

## Example Queries

- How many tracks are in the database?
- List 5 customers from India.
- Which artist has the most albums?
- Rank tracks based on listening time.
